Managing the day-to-day operations of an automotive dealership requires complex state management and reliable logic. To solve this, I developed **Automanage**, a core software application written in **Java**.

### Core Functionality
This application handles the critical business logic needed to run a dealership floor:
* **Inventory Tracking:** A robust system for cataloging vehicles, managing stock levels, and tracking metadata for individual cars.
* **Appointment Scheduling:** Logic to handle customer bookings, ensuring sales representatives and mechanics are scheduled efficiently without overlapping conflicts.

### The Road Ahead
Automanage serves as the foundational logic layer for a much larger project I am currently developing: a full-stack Dealership Management System (DMS). The goal is to evolve this Java backend into a comprehensive web platform featuring an integrated AI Chatbot for customer inquiries, automated profit/loss calculations, tax tracking, and secure dealer admin portals.
